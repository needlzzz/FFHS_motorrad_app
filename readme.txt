***** Service Worker assignment *****

I've implemented 2 versions of the service worker.

1. A Service Worker that caches specific pages (sw_cached_pages.js). This makes sense, when the website doesn't contain that many pages.
2. A Service Worker that caches the requests that the user makes - e.g. when the user clicks on a page, the page gets cached and is available offline. (sw_cached_wholesite.js)
